# How-to run your own SQL Fiddle

## Run docker-compose

`docker-compose -f docker-compose.yml up`

## Open PgAdmin

1. Go to http://localhost:5050/
2. Login with credentials from `env` file.

Username: `pgfiddle@localhost`

Password: `pgfiddle`
